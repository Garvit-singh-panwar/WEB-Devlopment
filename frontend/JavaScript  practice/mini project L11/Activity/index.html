Event Delegation

when we add a new todo's our delete function did not recognize it or do not do any action on it

Same event listner do not apply on new list items

if we want that listner should also be applied on our new todo's we have to use event delegation

event delegation is a phenomenon in which we use our bubbling property

if we want something have to trigger for our child element we will write same thing for our parent element instead for writing listner for our parent element we write listner for our parent element

        // target gives the element on chich we are clicking 

        //node name gives the name of the element

        ul.addEventListener("click",function(event){
            if(event.target.nodeName == 'BUTTON')
            {
                let listItem = event.target.parentElement;     //to delete list we have to access the parent of button 
                listItem.remove();
            }
        })
