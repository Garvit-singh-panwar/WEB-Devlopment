Spread

Expands an interable into multiple value

        function func(...arr) {
            //do something  
        }

        //Example 
        Math.min(1,2,3); // gives us the min value but we can only pass value in it

        // if we store value in array we can pass like this

        let arr = [1,2,3,4,5,6];
        Math.min(arr[1] , arr[2] , arr[3]); // it will consume more time 
        //so instead of passing value like this we can pass value like
        Math.min(...arr); //... is spread function
        // it gives 1

Spread with array literals

        // we can make a copy of array using spread operator
        let arr = [1,2,3,4,5];
        let newArr = [...arr];
        // newArr = [1,2,3,4,5] 
        
        //if we want to breakdown sttrings into individual characters we can use spread operator

        let chars = [..."hello"];
        console.log(chars); //chars = ["h","e","l","l","o"

Spread with object literals

        let data = {
            email: "ironman@gmail.com",
            password: "abcd"
        };

        let datacopy = {...data , id: 123};
        // datacopy = {
            email: "ironman@gmail.com",
            password: "abcd"
            id: 123
        }; id :123 is the new property which we added in datacopy
