Different terms

  • Command line - any interface that is used by entering textual commands (gen.windows centeric)
  • terminal - this is a type of command line (gen .mac centeric)
  • Console - A command line interface used to work with your computer
  • Shell - A program running on terminal
  • Blash - A popular shell on Mac OS / Linux
  • Z-Shell - Another Shell (default)

basic commands
