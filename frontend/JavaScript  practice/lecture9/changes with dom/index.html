    Selecting Elements by class name

    getElementByClassName

    Returns the Elements as an HTML Collection or empty collection (if not found)

            document.getElementByClassName("oldImg");
        //Returns a collection of object
        //if we want to access a perticular object we can do it by selecting an object by its indx
        document.getElementByClassName("oldImg")[0];
        //it returns an object of class oldImg at 0 index

        let smallImage = document.getElementByClassName("oldImg");
        for(let i = 0 ; i < smallImage.length ; i++)
        {
            smallImage[i].src = "assets/spyderman_img.png";
            console.log(`vlaue of img no. $(i) is changed.`);

        }

    Selecting Element Using tagname

    getElementByTagName

    Returns the Element as an HTML Collection or empty collection (if notfound)

            document.getElementByTagName("p")[1].innerText = "abc";
        //Paragraph text changed into abc

    Query Selectors

    Allows us to use any CSS selector

            document.querySelector('p'); // Selects first p element 
        document.querySelector('#myId'); //Selects first element with id = myId
        document.querySelector('.myClass'); //Selects first element with class = myClass
        document.querySelectorAll("p"); //Selects all p elements
